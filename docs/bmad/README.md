# KOS2 BMAD Bootstrap

To repo is prepared for a BMAD-driven rewrite and productization pass around an Ollama-first Obsidian plugin.

## Current state

- `KOS2` is a soft fork of `obsidian-copilot`.
- Default runtime is now `Ollama local + Ollama Cloud web tools`.
- Legacy Brevilabs license flow is neutralized in the bootstrap path.
- BMAD seed artifacts live in this folder.

## Recommended BMAD flow

1. `bmad-bmm-create-prd`
2. `bmad-bmm-create-architecture`
3. `bmad-bmm-create-epics-and-stories`
4. `bmad-bmm-check-implementation-readiness`
5. `bmad-bmm-sprint-planning`

# Local Ollama Notes

- Local Ollama is the default runtime for KOS2 chat and embeddings.
- Ollama Cloud is optional and only used for web search or web fetch flows.
- macOS Keychain fallback item for cloud tools is `cos2-ollama-cloud`.
- Benchmarks should record cold chat, warm chat, and embedding latency.
